Open `index.html` in your browser (or visit the GitHub Pages URL) to access all tools.

## Setting up GitHub Pages

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Under **Source**, select `Deploy from a branch`
4. Select `main` branch, `/ (root)` folder
5. Hit **Save**

Your site will be live at `https://yourusername.github.io/your-repo-name/`

---

## Using the C4 with AI

The C4 works without an API key (uses built-in templates), but AI-generated captions require an Anthropic API key.

1. Get a key at [console.anthropic.com](https://console.anthropic.com)
2. Open the C4 → click **Add New Client** → paste your key in the API Key field
3. The key is saved to your browser's local storage — you only enter it once

**Cost:** ~$0.30–0.90 per client per month for a full 30-day calendar.
